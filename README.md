# ⚡ Tank World — 局域网联机坦克大战

局域网实时双人坦克对战游戏，基于 WebSocket 同步 + UDP/HTTP 自动发现，同网段设备无需手动输入 IP 即可加入。

## 快速开始

```bash
# 安装依赖
npm install

# 启动服务器
node server.js
```

启动后会显示：
- **游戏端口**: 3000 (HTTP + WebSocket)
- **UDP 广播端口**: 3001 (自动发现)
- **HTTP 发现端口**: 3002 (扫描发现)

### 局域网联机

1. 主机运行 `node server.js`
2. 其他设备浏览器打开 `http://<主机IP>:3000`
3. 点击 **🔍 扫描局域网服务器** 按钮，自动发现同网段服务器
4. 点击发现的服务器即可连接

### 手动连接

如果自动发现失败，可手动输入服务器 IP 地址连接。

## 操作

| 操作 | 按键 |
|------|------|
| 移动 | `W` `A` `S` `D` |
| 射击 | `空格` |
| 重新开始 | `R` |

## 架构

```
server.js   ← 游戏服务器 (WebSocket :3000 + UDP 广播 :3001 + HTTP 发现 :3002)
index.html  ← 客户端 (渲染 + 输入采集 + 局域网扫描)
```

- 所有碰撞、伤害、胜负判定在服务端完成
- 客户端每帧发送输入，接收状态同步
- 子弹可反弹墙壁，有障碍物掩体
- **UDP 广播**: 服务器每 2 秒向局域网广播自身信息
- **HTTP 发现**: 客户端扫描局域网 IP 段的 3002 端口查找服务器

## 局域网发现机制

服务器启动后自动运行两个发现服务：

1. **UDP 广播** (端口 3001): 每 2 秒向 `255.255.255.255:3001` 广播 JSON 消息
2. **HTTP 发现接口** (端口 3002): 响应 `GET /discover` 请求，返回服务器信息

客户端点击"扫描"按钮时，会并行扫描当前子网及常见子网（192.168.0.x、192.168.1.x）的 3002 端口，发现服务器后自动显示在列表中。
